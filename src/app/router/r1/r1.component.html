路由定义配置（无参）


1. 生成路由模块,并配置到app模块

ng g module app-routing --module=app


2. 配置路由模块

import { Routes, RouterModule } from '@angular/router';

@NgModule ({ imports: [ RouterModule.forRoot ( appRoutes ) ], exports: [RouterModule] } )


2. 定义预加载的组件（引入该组件）

import { XxComponent } from './xx/xx.component';

const routes: Routes = [ { path: 'xx', component: XxComponent } ];


2. 定义惰性加载的模块（不引入）

const routes: Routes = [{
path: 'xx', loadChildren: './xx/xx.module#XxModule'
}];


3. 使用

< router-outlet >< /router-outlet >

[routerLink]="['/a']"


路由定义配置（传参）

1.定义

{ path: 'xx/:id', component: XxComponent }

2.使用

[routerLink]="['/a/hero.id']"

3.获取路由中的参数(必选参数？可选参数？)

const id = +this.route.snapshot.paramMap.get ('id')
路由参数总会是字符串。 JavaScript 的 (+) 操作符会把字符串转换成数字，英雄的 id 就是数字类型。
