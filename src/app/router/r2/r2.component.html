惰性模块（路由懒加载）


1.1. 生成带路由的特性模块

ng g module A -routing


1.2. 定义该模块

app-routing模块中：{ path: 'a', loadChildren: './a/a.module#AModule' }


1.3. 路由出口及链接

app.html中：
< router-outlet > </router-outlet>
[routerLink]="['/a']"


2.1. 生成特性模块的组件

ng g c 组件名


2.2. 定义该组件的路由

{ path: 'a1', component: A1Component }


2.3. 路由出口及链接

[routerLink]="['/a/a1']"
