使用管道:

在html中：

  • 1. 普通使用： { { name | date } }
  • 2. 链式管道： { { name | date | uppercase } }
  • 3. 传参管道： { { name | date: "fullDate" } }

内置管道

管道 用途
date 格式化时间 不推荐 date:'y/m/n' （多语言会死）=> date: 'shortDate'
json 转换成json字符串，缩进两个空格
async 把可订阅的数据转换成数据，然后放到界面上显示
slice 截取字符串 slice : x : y， -- {{ '123456' | slice: 1:4 }}
uppercase 大写
lowercase 小写
titlecase 首字母大写 -- {{ 'happy coding ' | titlecase }}
number 格式化数字 -- {{ 1 | number: '3.0' }} {{ 1 | number: '1.3' }}
percent 百分比转换 -- {{ 0.2 | percent }}
currency 货币转换 -- {{ 80 | currency }}
i18n 多语言转换 -- 参看i18n模块
