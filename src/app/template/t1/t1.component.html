指令分类

1. 组件 拥有模板的指令。

2. 结构型指令 改变DOM元素的结构。

3. 属性型指令 改变DOM元素的外观、行为。


结构型指令

a. 内置结构型指令

NgIf

< div *ngIf="hero" class="name">{ { hero.name } }< /div>

< ng-template [ngIf]="hero"> < div class="name">{ { hero.name } }< /div > < /ng-template>

NgFor *ngFor="let hero of heroes"

NgSwitch [ngSwitch]="hero?.emotion" *ngSwitchCase="'happy'" [hero]="hero"

星号(*)语法糖，会变成一个 < ng-template > 标记， 包裹着宿主元素及其子元素。

在一个宿主元素上,只能有一个结构型指令，但可以有多个属性型指令。
