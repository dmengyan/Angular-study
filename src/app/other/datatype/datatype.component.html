TypeScript 中的 数据类型：


1. 布尔值 Boolean

注意，使用构造函数 Boolean 创造的对象不是布尔值,

事实上 new Boolean() 返回的是一个 Boolean 对象


2. 数值 Number

和JavaScript,TypeScript里的所有数字都是浮点数。
js支持十进制和十六进制字面量，Typescript还二进制和八进制字面量。


3. 字符串 String

其中 ` ` 用来定义 ES6 中的模板字符串，${ } 来嵌入表达式。


4. 空值 Void

JavaScript 没有空值（Void），在 TypeScirpt 中，void 表示没有任何返回值的函数。


5. Null 和 Undefined

  let u: undefined = undefined;  // undefined 只能 = undefined
  let n: null = null;  // null 只能 = null
  let u: undefined;
  let num: number = u;  //undefined 类型的变量，可以赋值给 number 类型的变量

与 void 的区别是，undefined 和 null 是所有类型的子类型。


6. 任意值 Any

变量声明时，未指定类型，默认为 Any。

声明为任意值的变量，返回的内容的类型也是任意值。

  let notSure: any;
  notSure = 666;

  let list: any[] = [1, true, "free"];
  list[1] = 100;

7. 对象类型 Object 【非原始类型】

使用object类型，就可以更好的表示像Object.create这样的API。

declare function create(o: object | null): void;

create({ prop: 0 }); // OK
create(null); // OK

create(42); // Error
create("string"); // Error
create(false); // Error
create(undefined); // Error

8. 数组 Array

有两种方式可以定义数组:

  方法1：
  let list: number[] = [1, 2, 3];
  方法2：
  let list: Array< number> = [1, 2, 3];


7. 元组 Tuple

数组 : 合并相同类型的对象 ;
元组 : 合并不同类型的对象。

tuple: [string, number, boolean] = ['12345', 123, true , 666, 'lalala' ];

红色的是越界的元素，类型是联合类型，即 string | number | boolean，多种类型中的一种。
如果一个值是联合类型，我们只能访问此联合类型的所有类型里共有的属性或方法。


8. 枚举 Enum

定义一些带名字的常量。 清晰地表达意图或创建一组有区别的用例。
TypeScript支持数字的和基于字符串的枚举

数字枚举
enum Direction {
    Up = 1,  // 不设初始值的话 = 0
    Down,    // 依次 + 1
    Left,
    Right
}
字符串枚举
enum Direction {
    Up = "UP",
    Down = "DOWN",
    Left = "LEFT",
    Right = "RIGHT",
}
常数项和计算所得项
区分 常数项 计算所得项
1.定义 常数或者通过一元二元计算得到的 类似异步得到的，比如：Red = "red".length
2.顺序 随意 不能放在常数项前面，会报错
常数枚举

常数枚举与普通枚举的区别是，它会在编译阶段被删除，并且不能包含计算成员。

const enum Direction {
    Up,
    Down,
    Left,
    Right,
}
外部枚举

declare 定义的类型只会用于编译时的检查，编译结果中会被删除。

declare enum Direction {
    Up,
    Down,
    Left,
    Right,
}
使用时：
console.log(Direction.up, Direction.down, Direction.left, Direction.right);

类型推论

定义时没赋值，之后都会被推断成 any 类型，不检查类型。

let myFavoriteNumber;
myFavoriteNumber = 'seven';
myFavoriteNumber = 7;   // 此时正常

定义时赋值, 会推测数据的类型，之后不能改。

let myFavoriteNumber = 'seven';   // 类型推测为 string
myFavoriteNumber = 7;   // 此时报错

联合类型的变量在被赋值的时候，会根据类型推论的规则推断出一个类型:

let myFavoriteNumber: string | number;
myFavoriteNumber = 'seven';
console.log(myFavoriteNumber.length); // 5
myFavoriteNumber = 7;
console.log(myFavoriteNumber.length); // 编译时报错
